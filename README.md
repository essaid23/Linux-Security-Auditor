# Linux Security Auditor & Network Diagnostic Tool

A lightweight Linux automation and hardening script designed for system diagnostics and network security compliance auditing.

## 🚀 Features
- **System Diagnostics:** Monitors CPU, memory usage, and running services.
- **Network & ACL Auditing:** Verifies router Access Control List (ACL) configurations and open ports.
- **Security Audit:** Scans for potential vulnerabilities, misconfigured permissions, and analyzes system authentication logs.
- **Automated Reporting:** Generates a structured log file with diagnostic alerts.

## 🛠️ Tech Stack
- **Scripting:** Bash Shell, Python
- **Environment:** Kali Linux / Ubuntu Server
- **Tools:** Network auditing utilities, standard Linux security modules

## 📦 Installation & Usage
```bash
# Clone the repository
git clone [https://github.com/essaid23/Linux-Security-Auditor.git](https://github.com/essaid23/Linux-Security-Auditor.git)

# Navigate to the project directory
cd Linux-Security-Auditor

# Make the script executable
chmod +x auditor.sh

# Run the auditor
./auditor.sh
